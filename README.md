# nvidia-funcontrol
nvidia-funcontrol widget for KDE

виджет:

Управляет вентиляторами через sudo-скрипт

Реагирует на изменения из nvidia-settings (а не только из виджета)

Показывает температуру и скорость в тултипе

Умно моргает при перегреве, если забыл переключить режим

Правильно работает с турбинами (учитывает минимальные 30%)

Опрашивает GPU каждые 5 секунд

Имеет установочный пакет

Проверил на rtx pro.


Автоматизацию управления не стал прикручивать, может плачевно закончиться )
